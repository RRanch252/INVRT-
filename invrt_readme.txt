# INVRT Website

A clean, sophisticated website for INVRT, a strategic advisory firm focused on building enduring enterprises through systems thinking.

## Project Structure

```
invrt-website/
├── css/
│   └── styles.css
├── js/
│   └── script.js
├── index.html
├── philosophy.html
├── clients.html
├── insights.html
├── advisory.html
├── contact.html
└── README.md
```

## Features

- Minimalist, sophisticated design that aligns with the INVRT brand
- Responsive layout that works on all device sizes
- Multi-page structure with clear navigation
- Subtle animations and visual enhancements
- Alternating section backgrounds for visual rhythm
- Forms for contact and newsletter subscription

## Pages

- **Home (index.html)**: Minimalist homepage with hero section and brief introduction
- **Philosophy**: Our approach and four core principles
- **Clients**: Who we work with and client stories
- **Insights**: Latest thinking and podcast episodes
- **Advisory**: Detailed service offerings
- **Contact**: Contact form and newsletter subscription

## Technology

This website uses:
- HTML5
- CSS3 (via Tailwind CSS CDN)
- Vanilla JavaScript (no frameworks)

## Setup

1. Clone this repository:
```
git clone https://github.com/yourusername/invrt-website.git
```

2. Open any of the HTML files in your browser to view locally

3. To deploy, upload all files to your web hosting service

## Customization

- Edit the HTML files to update content
- Modify styles.css to change the visual appearance
- Update script.js for interactive functionality
